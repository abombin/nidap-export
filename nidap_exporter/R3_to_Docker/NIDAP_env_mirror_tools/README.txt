This repo is to hold any scripts associated with NIDAP migration.

To setup this repo:

 1-Add the environment variable "NIDAP_SCRIPTS" in your .bashrc file and point it to the directory of this folder
 2-Edit your PATH to:
    export PATH=$PATH:$NIDAP_SCRIPTS/scripts
 3-Create a file ~/.nidaprc and set in it the NIDAP developer token:
  nidap_token=<your NIDAP token>
  You can generate the NIDAP developer token in your user account settings on NIDAP
 4-Add the environment variable "NIDAP_SCRIPTS" in your .nidaprc file and point it to the directory of this repo
